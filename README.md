# はばタンPay 周辺マップ (非公式)

このプロジェクトは、はばタンPayの周辺にある飲食店やスーパー等を地図上に表示するためのWebアプリケーションです。

## 機能

1. **現在地取得**: ユーザーが現在地を取得し、その近くの店舗を表示します。
2. **店名検索**: 店名で店舗を検索できます。
3. **距離フィルタリング**: 一定距離内の店舗のみを表示できます。

## 使用方法

1. ブラウザで `index.html` を開きます。
2. 現在地を取得するボタンをクリックします。
3. 検索欄に店名を入力して、該当する店舗を検索します。

## 注意事項

- 本アプリは非公式です。最新情報は公式サイトをご確認ください。
- 地図データの利用には [OpenStreetMap](https://www.openstreetmap.org/) を使用しています。
