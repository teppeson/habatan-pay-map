# はばタンPay 周辺マップ (非公式)

はばタンPay＋（プラス）の利用可能店舗を地図上で簡単に探せる、個人開発の非公式Webアプリです。
現在地周辺の店舗（中央区・兵庫区の飲食店等）を素早く見つけることができます。

## 特徴
- 📍 **地図表示**: LeafletとOpenStreetMapを使用したスムーズな地図操作。
- 🔍 **フィルタリング**: 店名検索、距離指定（500m〜5km）、カテゴリ絞り込み（カフェ・レストラン）。
- 📱 **モバイル最適化**: iPhoneなどのスマートフォンで使いやすいデザイン。

## ローカルでの確認方法
このリポジトリをクローンまたはダウンロードし、ブラウザで `index.html` を直接開くか、ローカルサーバー（VSCodeのLive Server等）を起動して確認してください。

※ 位置情報の取得にはHTTPS環境、または `localhost` 環境が必要です。

## GitHub Pagesでの公開手順
1. GitHubに新しいリポジトリを作成します。
2. このプロジェクトのファイルをリポジトリにアップロード（またはpush）します。
3. リポジトリの `Settings` > `Pages` に移動します。
4. `Build and deployment` > `Branch` で `master` ブランチを選択して保存します。
5. 数分後、`https://<ユーザー名>.github.io/<リポジトリ名>/` で公開されます。

## iPhoneでの確認方法
1. 公開されたURLにSafariからアクセスします。
2. 「現在地を取得」ボタンを押し、位置情報の利用を許可してください。
3. ホーム画面に追加すると、アプリのように利用できます（共有ボタン > ホーム画面に追加）。

## 注意事項
- **非公式アプリ**: 本アプリは「はばタンPay＋」の公式アプリではありません。
- **データについて**: 掲載されている情報は作成時のものであり、最新の状況と異なる場合があります。必ず公式サイトの最新情報を確認してください。
